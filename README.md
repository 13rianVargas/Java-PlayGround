## 📦 Instalación y Uso

### Clonar el repositorio

```bash
git clone https://github.com/13rianVargas/Java-PlayGround.git
cd Java-PlayGround
```

### Ejecutar proyectos Spring Boot

```bash
# Navegar al proyecto deseado
cd "Java Spring Framework/control_horario"

# Compilar el proyecto
./mvnw clean install

# Ejecutar la aplicación
./mvnw spring-boot:run
```

### Compilar y ejecutar archivos Java básicos

```bash
# Navegar al directorio del código fuente
cd "Java Developer Roadmap/01-Learn the Basics/01-Basic Syntax/src"

# Compilar
javac App.java

# Ejecutar
java App
```
